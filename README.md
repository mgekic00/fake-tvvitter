# fake-tvvitter

a simple React application to showcase some good practices. Includes a very poorly written backend just to go along :)
BE requires Maven.

RUN:

cd fake-backend  
mvn clean install -DskipTests  
mvn spring-boot:run    


cd gui  
npm install  
npm start      


Usage:  
Register a new user and login.  
Search for "user" using the search bar and follow user1 and user2 to see some posts :)
